## flask demos with some flask extensions

## To get started

* git clone code
* Create a virtual environment with *pipenv*
> $ pipenv --python 3.7
* install packages
> $ pip install -r requirements.txt


## Run flask-appbuilder
> $ pipenv shell
> $ cd appbuilder
> $ python run.py

## Run flask-babel
> $ pipenv shell
> $ cd babel
> $ python run.py

## Run flask-login
> $ pipenv shell
> $ cd login
> $ python run.py

## Run flask-socketio
> $ pipenv shell
> $ cd socketio
> $ python run.py

## Run flask-sqlalchemy
> $ pipenv shell
> $ cd sqlalchemy
> $ python run.py
